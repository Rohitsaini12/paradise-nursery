Paradise Nursery is a React-based e-commerce web application for an online plant shop.  
Users can browse different categories of houseplants, view plant details, add items to a shopping cart, and manage quantities dynamically.

## Features
- Browse plants by categories
- Add plants to cart
- Increase or decrease quantity
- Remove items from cart
- View total cost dynamically
- Simple and user-friendly UI

## Tech Stack
- React.js
- Redux Toolkit
- CSS
